Features

Sign up and Log in
Teachers can assign tasks to students
Teachers can edit names in Settings page.
Students can submit tasks
Students can see all previous and upcoming tasks in the calendar
Teachers and Students can remove tasks in Settings page



Title: Rules

1. Fork the GitHub repository and deploy it to Vercel before the race begins
    → Copy (fork) the given project to your own GitHub account and deploy it on Vercel before starting.
2. A demo of the working website will be shown
    → You will see a fully working version so you know how the app should behave.
3. Only fix bugs in the existing code
    → You are allowed to fix errors only.
    ❌ No rewriting or rebuilding any part from scratch.
4. Do not add new UI, pages, or features
    → You cannot add anything that doesn’t already exist in the project.
5. Do not change design or layout
    → The look (UI, styling, layout) must stay exactly the same.
6. Your Vercel link must open the login page first
    → When someone opens your deployed link, it must show the login page immediately.
    ❌ If not → disqualified.
